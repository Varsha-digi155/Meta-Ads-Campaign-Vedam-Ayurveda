# 📱 Meta Ads Campaign – Vedam Ayurveda

A complete Meta (Facebook & Instagram) Ads campaign created for **Vedam Ayurveda**, an Ayurvedic skincare brand. This portfolio project demonstrates the end-to-end process of planning, creating, and configuring a Meta Traffic campaign designed to drive qualified visitors to a landing page.

---

# 📌 Project Overview

This project was created to gain practical experience with Meta Ads Manager by designing a campaign that follows industry-standard practices.

The campaign promotes the **Vedam Ayurveda Dosha Skin Guide** through Facebook and Instagram, encouraging users to visit the landing page and explore Ayurvedic skincare solutions.

---

# 🎯 Campaign Objective

- Increase website traffic
- Maximize landing page views
- Reach a relevant skincare audience
- Build brand awareness
- Create a portfolio-ready Meta Ads campaign

# ⚙️ Campaign Configuration

| Setting | Details |
|---------|---------|
| Platform | Meta Ads Manager |
| Campaign Objective | Traffic |
| Buying Type | Auction |
| Conversion Location | Website |
| Performance Goal | Maximize Landing Page Views |
| Budget | ₹800/day |
| Audience | Women (25–45 Years) |
| Locations | Tier-1 Indian Cities |
| Ad Format | Single Image |
| Landing Page | Google Sites |

# 📊 Challenges Faced

During campaign creation, several platform-related challenges were encountered and resolved, including:

- Meta Page Naming Restriction
- Creative & Objective Mismatch Error
- Carousel Format Validation
- Incorrect Ad Preview Display

These issues were successfully resolved by adjusting campaign settings according to Meta Ads Manager best practices.
